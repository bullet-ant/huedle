# Huedle

A color-guessing puzzle game inspired by Wordle. Crack the secret 4-color sequence in 7 attempts or fewer.

## How to Play

1. Guess the hidden sequence of 4 colors
2. Each color (Blue, Green, Orange, Yellow) appears exactly once
3. After each guess, dots show how many colors are in the correct position
4. Solve it in as few guesses as possible

## Controls

- **Click** on a slot, then pick a color from the palette
- **1-4 keys** to quickly select colors
- **Arrow keys** to navigate between slots
- **Backspace** to clear
- **Enter** to submit

## Tech Stack

- Svelte + TypeScript
- Vite
- CSS (no frameworks)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
